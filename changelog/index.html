    Ocean 0.2 RELEASE! 🌊

    We’ve made a significant number of changes since releasing the first version of Ocean Beta 0.1.
    Below is a summary of the updates we consider the most important:
    Starting today, the only official version of Ocean will be hosted at https://anticheat.ac. The previous dashboard (used since ~2023) is now fully deprecated and will no longer receive updates or support.

    📁 Database Acquisition: SecuroX, Analystic & Screenshare.lol

    We’ve launched a new partnership with the following intelligence platforms:
    • SecuroX — https://inspect.wtf/
    • Screenshare.lol — https://screenshare.lol/
    • Analystic Network — https://analystic.de/
    These tools are specialized in gathering and analyzing data on suspicious users, cheaters, and “raiders” across various communities and regions.
    All collected intelligence is processed and delivered directly to Ocean, providing detailed and reliable reports on user activity and potential threats.

    💳 Integrated Subscription Payments & Crypto Support

    Ocean now fully supports subscription-based payments through our website:
    https://anticheat.ac/pricing
    In addition, we’ve introduced automatic cryptocurrency payment support, allowing faster and more flexible access to your licenses.
    You can view and manage all your payments and active subscriptions, including the ability to cancel them at any time, through the following link:
    https://anticheat.ac/dashboard/settings?tab=billing
    With this upgrade, you no longer need to submit a payment receipt to receive your license.
    If you complete your purchase using the same email you used to register your Ocean account, your license will be granted automatically.
    Note: If you used a different email during checkout, you will need to open a support ticket.

    🧩 Minecraft Free License Changes

    As many of you know, Ocean provides free Minecraft scans.
    This free tier has now been limited to 5 PINs per day to reduce abuse from testers, bypassers, and users taking advantage of the unrestricted access previously offered.
    To remove this limitation, you can upgrade to Ocean+ (the dedicated Ocean Minecraft license) or apply for media access via ticket.

    🖥пёЏ Updated GUI & CLI

    As many of you know, the Ocean GUI had begun to feel outdated — so we’ve delivered a complete visual overhaul.
    Both the GUI and the CLI have been redesigned to give Ocean a cleaner, more modern interface.
    The OceanCLI download URL has also been updated:
    https://api.anticheat.ac/dl/cli
    Functionality remains the same: the CLI continues to provide a clear, direct view of what’s happening inside Ocean in real time.
    Additionally, we’ve introduced a new command-line argument:
    nodesign — Use this flag to disable the CLI’s visual design and display only percentage-based progress for a cleaner, minimal output.

    🔌 Integration of Our Own API

    Ocean has developed its own internal API to provide global results for users previously scanned with our system.
    This allows you to retrieve clear and consistent detection histories for any individual scanned through Ocean.
    A dedicated customer-only API is now available, allowing you to query a user’s detection history using their Discord ID.
    You can view the full documentation here:
    https://anticheat.ac/docs/api
    This API also provides insight into any alternate accounts (alts) associated with the user.

    BETA Release 0.1

    v0.1

    28 October 2025chicho

    🌊 Ocean BETA Release 0.1

    A little bit of what our year of work was like to release the beta of Ocean.
    For over a year, we have been considering the idea of creating a new dashboard. The previous version, in addition to having several issues, was built on a language that, at the time, provided limited stability and was not sustainable in the long term.
    With this in mind, we are proud to present a completely reimagined Ocean—built from the ground up. It retains all the core foundations of the old Ocean, while embracing modern standards and the maintainability that the project truly deserves.

    🧠 Ocean Beta Release Notes

    🟢: Addition | 🔵: Improvement | 🔴: Deletion
    1. UnTrusted Files System 🟢
      Introduced UnTrusted Files, a powerful new system designed to identify both indexed and non-indexed files using Ocean’s extensive and continuously evolving detection database.
    2. Integrity Check System 🟢
      Added advanced integrity verification to detect file discrepancies, tampering, and anomalies in critical system files.
    3. System Hooking Detection 🟢
      Implemented detection for system-level hooking techniques commonly used to bypass security mechanisms or alter system behavior.
    4. AI-Powered Integrity Checks 🟢
      Introduced integrity checks powered by artificial intelligence to analyze behavioral patterns and identify advanced tampering strategies.
    5. UEFI/EFI Detection 🟢
      Ocean is now the first screenshare tool to detect tampering in UEFI and EFI environments, including:
      • UEFI Modifications
      • Tampered EFI Files
      • Booting Discrepancies
      • Malicious Boot Alterations
    6. RUIN Mode 🟢
      Launched RUIN Mode, built using similar methods to internal ghost clients—designed to catch even the most subtle manipulations within the game’s internal memory space.
    7. Arduino & Raspberry Pi Detection 🟢
      Ocean now detects external devices such as Arduino and Raspberry Pi, including spoofed hardware variants and AIM devices.
    8. Video Fuser Detection 🟢
      Added capabilities to detect the presence of video fusion software commonly used to manipulate or alter visual evidence.
    9. Mapped & Vulnerable Driver Detection 🟢
      Expanded driver analysis to include mapped drivers and known vulnerable drivers that could be leveraged for exploits.
    10. Sysmon Support 🟢
      Ocean now supports Sysmon integration, enabling deeper monitoring and correlation with system activity.
    11. Ocean Documentation 🟢
      Released official documentation for Ocean, helping users better understand configuration, scanning behavior, and reporting.
    12. Expanded Anti-Tamper & Integrity Checks (20+ New) 🟢
      Added over 20 new anti-tamper and integrity validation routines, covering areas such as:
    • Network bypass detection
    • Process injection analysis
    • Impossible and suspicious file operations
    • RAR file execution detection
    • Forensic artifact tampering
    • Executed, deleted, or modified files
    • Invalid PE signatures
    • Fake GUI detection
    • Screenshot residue identification
